# Стабильная поддержка

Репозиторий документов по проекту [благотворительного фонда «Внуки»](https://vnuki.fund): методические и образовательные материалы, актуальные справки, агрегированные данные, релевантные исследования, правовая информация — всё необходимое для работы по проекту.

## Навигация

- [Методические материалы](Методические%20материалы/) — описание проекта, роли, модель, методички для волонтёров и координаторов.
- [Документы, соглашения, памятки](Документы,%20соглашения,%20памятки/) — анкеты, соглашения с подопечными и волонтёрами, чек-листы, справки, структура отчётов.
- [Исследования](Исследования/) — обзоры и данные по социальному одиночеству и целевой аудитории.

В корне: [Социальная франшиза «Стабильная поддержка»](Социальная%20франшиза%20«Стабильная%20поддержка»_%20системная%20помощь%20пожилым%20людям.md) — сводный обзор проекта. [Беклог задач](BACKLOG.md) — отложенные задачи по документации.
