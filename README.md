# Стабильная поддержка

## Использование документов

Все документы в репозитории доступны для скачивания и свободного использования при условии **указания автора** — благотворительного фонда «Внуки».

Материалы распространяются под лицензией **[Creative Commons Attribution-NonCommercial 4.0 International (CC BY-NC 4.0)](https://creativecommons.org/licenses/by-nc/4.0/deed.ru)**. Вы можете свободно копировать, распространять и адаптировать материалы **только в некоммерческих целях** с обязательным указанием авторства и ссылки на источник. Для использования в коммерческих целях необходимо обратиться к правообладателю: **info@vnuki.fund**.

### Присоединение к методологии «Стабильная поддержка»

Чтобы присоединиться к работе по методологии «Стабильная поддержка», свяжитесь с фондом: **info@vnuki.fund**.

Франшиза бесплатна и даёт партнёру:

- постоянное обучение волонтёров и координаторов проекта;
- доступ к ресурсам партнёров, в том числе для организации адресной помощи благополучателям;
- право на использование бренда, презентаций и рекламных материалов проекта;
- обмен опытом в сообществе единомышленников.
