# Курс для координаторов

## Правила именования файлов (единый стандарт для обоих курсов)

- Формат: `NN_Тип_Краткое_название.ext`
- `NN` — двузначный порядковый номер (`01`, `02`, ...).
- Разделитель слов — `_`, без пробелов в имени файла.
- Для комплектов уроков в текстах: `NN_Уроки_X-Y.md`.
- Для аудиоуроков: `NN_Урок_N_Краткое_название.mp3`.
