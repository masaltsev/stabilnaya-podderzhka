# Курс для координаторов

Курс помогает координаторам выстроить устойчивую систему сопровождения: от понимания модели программы и подбора волонтёров до супервизии, работы со сложными случаями и управленческой отчётности. Цель обучения — дать единый стандарт действий, снизить риски сбоев в парах и повысить качество поддержки благополучателей. После прохождения координатор получает понятный рабочий маршрут и набор практик, которые позволяют уверенно вести команду и развивать локальное сообщество.
