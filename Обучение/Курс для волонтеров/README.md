# Курс для волонтеров

Курс знакомит волонтёров с принципами сопровождения пожилых людей: как выстраивать доверительный контакт, соблюдать границы, действовать безопасно и поддерживать регулярность общения. Его задача — превратить мотивацию помогать в устойчивую практику, где есть понятные роли, алгоритмы и поддержка координатора. После прохождения волонтёр лучше понимает, как помогать бережно и эффективно, не выгорать и быть для подопечного надёжным человеком на постоянной основе.
